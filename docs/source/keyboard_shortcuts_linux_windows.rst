====================================
Keyboard Shortcuts - Linux & Windows
====================================

General
-------

+----------------+-------------------------------------------+
| Keypress       | Command                                   |
+================+===========================================+
| ``Ctrl + O``   | Open capture file                         |
+----------------+-------------------------------------------+
| ``Ctrl + E``   | Interrupt loading of a large capture file |
+----------------+-------------------------------------------+
| ``Ctrl + S``   | Save current capture to file              |
+----------------+-------------------------------------------+

Live Capture
------------

+---------------+--------------------------+
| Keypress      | Command                  |
+===============+==========================+
| ``Ctrl + B``  | Begin live capture       |
+---------------+--------------------------+
| ``Ctrl + E``  | End live capture         |
+---------------+--------------------------+
